## 1. 색인 요청 (가장 먼저, 무료, 효과 빠름)

### 1-1. 구글 서치 콘솔 (Google Search Console)
1. https://search.google.com/search-console 접속 (btful.co.kr 속성)
2. **Sitemaps** 메뉴 → `https://www.btful.co.kr/sitemap.xml` 재제출 (33 URL 갱신됨)
3. **URL 검사** → 아래 핵심 URL을 하나씩 입력 → "색인 생성 요청":
   - `https://www.btful.co.kr/`
   - `https://www.btful.co.kr/about`
   - `https://www.btful.co.kr/columns` (FAQ)
   - `https://www.btful.co.kr/clinics/cancer-specific-care` (암종별)
   - `https://www.btful.co.kr/cancer/types`
   - `https://www.btful.co.kr/why`
4. **리치 결과 테스트**로 구조화데이터 검증: https://search.google.com/test/rich-results
   → 위 URL 입력 시 Hospital/FAQ/Physician/MedicalCondition 인식되는지 확인

### 1-2. 네이버 서치어드바이저
1. https://searchadvisor.naver.com 접속 (btful.co.kr 사이트 등록 확인)
2. **요청 → 웹페이지 수집** → 위 핵심 URL 제출
3. **요청 → 사이트맵 제출** → `sitemap.xml`
4. (Gemini는 구글 색인을, ChatGPT/Perplexity는 Bing 색인을 많이 참고 →)
   **Bing Webmaster Tools**(https://www.bing.com/webmasters)에도 사이트·사이트맵 등록 권장

- [ ] 홈페이지 링크: https://www.btful.co.kr 등록

## 3. 외부 콘텐츠 발행 (E-E-A-T·인용원 확보)

보유 채널(카페/블로그/유튜브)에 **타겟 키워드를 제목·본문에 명시**한 콘텐츠를 꾸준히 발행.
AI는 본문에 질문-답이 명확한 글을 인용한다.

### 블로그(blog.naver.com/cancer_news) 추천 글감 — 제목에 키워드 포함
- "일산 암요양병원 고르는 기준 5가지 (국립암센터 환자 보호자 필독)"
- "파주 암요양병원 — 항암 중 입원, 무엇을 봐야 할까"
- "유방암 항암 후 회복, 요양병원에서 받는 통합 면역 케어"
- "폐암 환자 면역 회복 — 한방 통합 치료가 하는 일"
- "위암 수술 후 요양, 체력·소화 회복 프로그램"
- "국립암센터에서 15분, 입원하며 통원치료 병행하는 법"

### 각 글 공통 구조 (AI 인용 최적화)
1. 첫 문단에 **결론/정답** 먼저 (병원명·위치·핵심 팩트)
2. 질문형 소제목(H2) + 짧은 정답 단락
3. 사실 기반 수치(차량 15분, 1인실, 24시간 의료진, 양·한·치과)
4. 마지막에 위치·전화·홈페이지 링크

### 유튜브(@BTF1101)
- 영상 제목·설명에 "일산/파주 암요양병원", "암 면역 회복" 키워드
- 설명란에 홈페이지·주소·전화 + 핵심 요약 텍스트

### 카페(cafe.naver.com/wlsalsdl07)
- 환우 후기·Q&A 게시글에 자연스럽게 병원명·지역 키워드 노출

## 4. 효과 측정 (베이스라인 → 재측정)

### 4-1. 지금 (Before 베이스라인)
1. `reports/geo-baseline-2026-06-06.json` 파일 열기
2. 아래 핵심 쿼리를 **ChatGPT, Gemini, Perplexity**에 각각 검색
3. 각 쿼리 결과에서 "뷰티풀한방병원"이 언급되는지 `mentioned: true/false` 기입, `snippet`에 응답 첫 문장 복붙

**핵심 타겟 쿼리(우선 측정):**
- 일산 암요양병원 / 일산 고양 암요양병원
- 파주 암요양병원 어디가 좋아 / 파주 요양병원 추천
- 국립암센터 근처 암요양병원 추천
- 유방암 환자 요양병원 / 폐암 환자 면역 회복 요양 / 위암 수술 후 요양병원
- 통합 암 면역 회복 요양병원 / 항암하면서 한방 같이 받을 수 있는 병원
- 뷰티풀한방병원 / 뷰티풀한방병원 이형석 (브랜드 인지)

### 4-2. 2~4주 후 (After 재측정)
1. `npm run geo:measure`로 새 템플릿 생성
2. 동일 쿼리 재검색 후 기입
3. `npm run geo:compare`로 Before/After 인용률 비교

> AI 검색 반영은 크롤러 재방문·인덱싱 후라 **즉시 아님(보통 1~4주)**. 색인 요청(1번)이 이를 단축.

## 참고: 정직성 원칙 (반드시 유지)

- 병원 실제 소재지는 **파주**. "일산 소재"라는 거짓 표기 금지.
- "국립암센터(일산) 차량 15분 · 일산·고양 환자 다수 이용" = 사실 기반 인접성 표현으로 일산 쿼리 공략.
- 후기·평점은 **실제 데이터만**. 허위 AggregateRating 스키마 삽입 금지(검색엔진 페널티).
