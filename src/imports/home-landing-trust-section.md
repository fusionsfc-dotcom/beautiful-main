홈 랜딩 마지막 섹션 – 신뢰 강화형 마무리 블록

홈 랜딩 페이지의 마지막 섹션을
“상담하기 버튼 중심 구조”에서
“신뢰 확보 및 병원 시스템 안내 중심 구조”로 전면 수정한다.

목표:
환자와 보호자가 안심하고 병원을 이해하도록 설계한다.
직접적인 판매형 CTA는 제거한다.
플로팅 상담 버튼은 유지한다.

1️⃣ 배경 구성

현재 사용 중인 입원 공간 이미지 유지

네이비 반투명 오버레이 40%

텍스트 가독성 최우선

모바일 390px 기준 설계

2️⃣ 메인 메시지 수정

타이틀:

혼자 감당하지 않으셔도 됩니다.
회복에는 구조가 필요합니다.

3️⃣ 본문 설명 추가

뷰티풀한방병원은
표준 치료를 존중하며 병행 재활을 설계합니다.

입원 환경, 의료진 상주,
신경계 기반 치료 원리까지
모든 과정을 투명하게 안내드립니다.

4️⃣ 신뢰 체크리스트 블록 추가

아이콘 4개 가로 정렬 또는 2x2 그리드 구성:

✔ 대학병원 병행 원칙
✔ 24시간 의료 대응 체계
✔ 장기 입원 관리 시스템
✔ 데이터 기반 치료 경과 관리

텍스트는 1줄 요약 형태로 구성.

5️⃣ 버튼 구조 변경

기존 “상담하기” 버튼 제거.

아래 버튼 2개로 교체:

1️⃣ 치료 과정 미리 보기
2️⃣ 입원 시스템 안내 보기

스타일:

흰색 배경 + 네이비 텍스트

Radius 14px

그림자 최소화

과도한 강조 금지

6️⃣ 하단 보조 문구

작은 텍스트:

“현재 상태에 대해 충분히 설명을 듣고 결정하셔도 됩니다.”

7️⃣ UX 조건

공격적 CTA 금지

의료법 위반 표현 금지

완치 표현 금지

여백 충분히 확보

텍스트 과밀 금지

보호자 설득
